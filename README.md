# SDA Project
Reverse engineering of a medium-sized open source project. 

## What to put where?
- `doc` folder stands for *"documents"*: all pdfs, pptx, and silimar files go here.
- `img` folder stands for *"images"*: all images used in the project should go here.
- `res` folder stands for *"resources"*: all additional files that provide some value and could be used for reference go here (useful links, books, pdfs, slides…).
- `src` folder stands for *"source"*: all *raw* code files go here, such as `.md`, `.puml`, `...`.  
    * `deliverables`: this folder holds all `.md` documents to be delivered .
    * `diagrams`: this folder contains all `.puml` or other similar diagram related source code.
- `journals` in this folder you can find each member's journal.
- `tools` any tool, make files, bash scripts or similar which may be useful for the project.
